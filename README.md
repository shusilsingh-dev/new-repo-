# Smart Retail & Health Analytics System

This project is an integrated data science application that combines retail analytics and health analytics using Python.

## 📌 Overview
The system demonstrates key data science techniques such as:
- Data filtering
- Data cleaning
- Handling missing values (NaN)
- Regression analysis
- Statistical testing
- Data visualization (heatmaps, graphs)

## 🛒 Retail Analytics
- Revenue calculation
- Sales filtering
- Trend analysis using Linear Regression
- Sales relationship visualization

## 🏥 Health Analytics
- Disease prediction using Logistic Regression
- Analysis based on Age, Blood Pressure, and Cholesterol

## 📊 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📈 Key Features
- End-to-end data analysis pipeline
- Machine learning models for prediction
- Visualization for better insights
- Clean and structured dataset handling

## 📂 Project File
- `Smart_Retail_Health_Analytics.ipynb`

## 🚀 Conclusion
This project demonstrates how data science can be applied to solve real-world problems in both business and healthcare domains.
